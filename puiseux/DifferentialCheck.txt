x  = X/Z, y  = Y/Z

xp = x/y, yp = 1/y

xp = Y/X, yp = Z/X
xp = y/x, yp = 1/x

F  = x^4 + y^4 + 1
Fp = 1 + xp^4 + yp^4

(xp / (dFp / dyp)) dxp
(-xp / (dFp / dxp)) dyp
(-1 / 4 xp^2) dyp
(-x^2 / 4 y^2) (-1/x^2) dx
(1 / 4 y^2) dx
--> y

(yp ...) gives (yp / xp) times this,
which is --> 1

(1 ...) gives (1 / xp) times this,
which is --> x

So if
x --> a1, b1, c1
y --> a2, b2, c2
1 --> a3, b3, c3

then
xp --> a2, b2, c2
yp --> a3, b3, c3
1  --> a1, b1, c1
